Class: Storage

Storage

new Storage(name, configPath)

Storage instances handle a json file where Generator authors can store data.

Base instantiate the storage as config by default.

Parameters:
Name Type Description
name String

The name of the new storage (this is a namespace)
configPath String

The filepath used as a storage. .yo-rc.json is used by default
