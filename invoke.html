Mixin: actions/invoke

actions/invoke

Receives a namespace, and an Hash of options to invoke a given generator. The usual list of arguments can be set with options.args (ex. nopt's argv.remain array)

DEPRECATION notice: As of version 0.17.0, invoke() should usually be replaced by composeWith().
